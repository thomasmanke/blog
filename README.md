# My Web Site


### Clone Repository

```
git clone 
```

### Install dependencies

```
mamba env create -f env.yml 
```

### Render web site
```
#https://github.com/quarto-dev/quarto-cli/discussions/3977
export QUARTO_PYTHON=/Users/manke/miniconda3/envs/web/bin/python
quarto render
quarto preview
```
